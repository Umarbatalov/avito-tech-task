# Avito-tech-task solution

[Тестовое задание](https://github.com/avito-tech/brave-new-billing)

## Что было реализовано из дополнительного

1. Подготовить OpenAPI-спецификацию. ✅
2. Ограничить время жизни платёжной сессии 30 минутами. ✅
3. Добавить API-метод, который возвращает список всех платежей за переданный период. ✅
4. После успешной оплаты асинхронно отправлять HTTP-уведомление на URL магазина.
   URL для таких уведомлений передаётся магазином в запросе /register.

## Запуск проекта

Так как проект тестовый, в настройке окружения нет особого смысла.
Можно сразу запустить команду

```shell script
docker-compose up -d
```

## Документация

В репозиторий был добавлен Swagger UI для создания интерактивной документацией по спецификации OpenAPI.
Документация доступна по URL http://localhost/swagger/index.html
